Smart Executor

Smart Executor is a light-weight thread execution and thread-pools management framework. The goal of the framework is to provide controllable tasks execution environmet while keeping the executions under tight control and monitoring.

 maven 
<dependency>
	<groupId>org.clevermore</groupId>
	<artifactId>smartexecutor</artifactId>
	<version>1.0.8</version>
</dependency>

Smart Monitoring

SmartMonitoring is an open source Java Monitoring Platform with HTML (GWT) UI that monitors Java processes via JMX connection. The project might be extended to include more monitoring abilities (widgets), the basic once are provided with in a scope of a current project.
